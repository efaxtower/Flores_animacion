## 📝 Descripción

Proyecto de animación CSS pura que simula un jardín nocturno de tulipanes neón. Las flores florecen al hacer clic en un botón y se mecen suavemente con el viento, creando una atmósfera mágica y envolvente.

Este proyecto nació como una práctica personal para dominar:
- Animaciones con `@keyframes`
- Transformaciones 3D (`rotateX`, `rotateY`)
- Efectos de neón con `box-shadow` y `drop-shadow`
- Fondos complejos con `repeating-radial-gradient`

---

## ✨ Características

- 🌸 **Tres tulipanes animados** que florecen desde cero
- 💖 **Efecto neón rosa** con brillo difuminado
- 🌙 **Cielo estrellado** con efecto de desenfoque (`blur`)
- 🎬 **Botón interactivo** "Florecer" que inicia la animación
- 💬 **Frases aleatorias** que aparecen al florecer
- 🌿 **Pastos y hojas** que se mueven con el viento
- 📱 **Diseño adaptable** (unidades `vmin`)
